# 🚀 OrangeHRM Manual Testing Project

A professional **Manual QA Testing portfolio project** focused on validating core HR workflows in the OrangeHRM demo application.

This repository demonstrates a strong **QA mindset** through:

* Functional manual testing
* Positive and negative scenarios
* UI validation
* Business workflow coverage
